Events

A class that models a single Poisson process.

rate : float, int
Rate of occurrence of an event is unit time.

n_events(t: float, int) -> rv_discrete

Return the model for the number of events that occur in t units of time.

This is modelled \(N(t) \sim\) Poisson(\(\lambda t\)), where \(\lambda\) = self.rate.

waiting_time() -> rv_continuous

Return a model for the time between successive events.

This is the model \(T \sim M(\lambda)\), where \(\lambda\) = self.rate.

Example use

Assume that patients arrive at a hospital at a rate of 6/hour.

Initialise the model.

hospital_arrivals = pp.Events(6)  # per hour
print(hospital_arrivals)
Events(rate=6)

We can return the model the number of events in unit time (i.e., one hour) by calling n_events().

Return the probability that no patients arrive at the hospital in a one hour window.

hospital_arrivals.n_events().pmf(0)
0.0025

Pass an argument if you want to rescale the window. For example, passing t=0.5 will return the model for the number of events that occur in 0.5 hour (or 30 minutes).

Calculate the probability that two patients arrive in 30 minutes.

hospital_arrivals.n_events(0.5).pmf(2)
0.2240

Calling the waiting_time() method will return the model for the time between successive events.

What is the probability that the interval between arrivals exceeds 30 minutes?

hospital_arrivals.waiting_time().sf(0.5)
0.0498

MultivariateEvents

A class that models a multivariate Poisson process.

rates : dict
A dict that maps a kind of event to its rate of occurrence.

aggregated_rate : float, int (Property)
The aggregated rate of occurrence for all kinds of event.

props : dict (Property)
A dict that maps a kind of event to its proportion of all kinds of events.

@classmethod
from_props(props: dict, aggregated_rate: float | int) -> MultivariateEvents

Initialise and return an instance of MultivariateEvents using proportions rather than rates. Argument aggregated rate should be the combined rate of all kinds of event.

Precondtions

  • sum(values(props)) = 1
  • 0 < prop <= 1 for each prop in values(props)
  • aggregated_rate > 0

n_kind(n_star: int, kind: object) -> rv_discrete

Return a model for the number of events of the given kind, given n_star events took place.

This is the model N ~ B(n_star, self.props[kind]. This is modelled \(N \sim B\)(n_star, \(p\)), where \(p\) = self.props[kind].

Example use

The different types of call to a mobile phone company’s call centre can be modelled as independent Poisson processes. Calls from customers that wish to cancel their subscription occur at a rate of 20 per hour. Calls from customers who wish to make a complaints occur at a rate of 35 per hour. And calls from customers who wish to subscribe to the service occur at a rate of 22 per hour.

Initialise the model.

call_rates = {'cancel': 20,
              'complain': 35,
              'new': 22}
call_centre = pp.MultivariateEvents(call_rates)  # per hour

The different models can be accessed by indexing. This returns a instance of Event.

Return the probability that 30 customers call to cancel their subscription over the course of 90 minutes.

call_centre['cancel'].n_events(1.5).pmf(30)
0.0726

Models can be combined by listing the different kinds of events.

What is the expected time between customers wishing to cancel or complain about the service?

call_centre['cancel', 'complain'].waiting_time().mean()
0.0182

The aggregated model that describes all events can be accessed by either passing : or ... as an index.

What is the probability that less that 60 customers call in one hour?

call_centre[:].n_events().cdf(59)
0.0198

What is the probability that 15 customers called to cancel their account, given 40 customers called?

Note, this returns frozen instance of scipy.stats.binom.

call_centre.n_kind(40, 'cancel').pmf(15)
0.0361

Animals pass by a person’s window according to a Poisson process at a rate of 20 animals per hour. Of those animals, approximately 60% are birds, 30% are cats, and 10% are dogs.

Initialise the model.

props = {'birds': 0.6,
         'cats': 0.3,
         'dogs': 0.1}
passing_animals = pp.MultivariateEvents.from_props(props, 20)

Functions

get_cont_xs(rv: rv_continuous) -> so.Plot

def get_cont_xs(rv):
    return np.linspace(rv.ppf(0.01), rv.ppf(0.99), num=50)

get_disc_xs(rv: rv_discrete, step: int = 1) -> so.Plot

def get_disc_xs(rv, step=1):
    return np.arange(rv.ppf(0.01), rv.ppf(0.99), dtype=int, step=step)

plot_n_events(ne: rv_discrete, step: int = 1) -> so.Plot

def plot_n_events(nt, step = 1) -> so.Plot:
    """Plot the approximated probability distribution of N(t),
    the number of events in time.

    Pass step to reduce the number of events plotted, if the returned
    plot is cluttered.
    """
    ns = get_disc_xs(nt, step)
    return (
        so.Plot(x=ns, y=nt.pmf(ns))
        .add(so.Bar())
        .label(x='n', y='p(n)', title='Probability distribution')
    )

plot_waiting_time(wt: rv_continuous) -> so.Plot

def plot_waiting_time(wt) -> so.Plot:
    """Plot the approximate probability distribution of T, the waiting
    time between events.
    """
    ts = get_cont_xs(wt)
    return(
        so.Plot(x=ts, y=wt.pdf(ts))
        .add(so.Line())
        .label(x='t', y='f(t)', title='Probability distribution')
    )

Case-Control Studies With Three or More Exposures

M249
Statistics
Author

Liam Kelly

Published

September 5, 2022

Summary notes

Perform an epidemiological analysis on the results of a case-control study with more than categories of exposure.

Pre-conditions

A cohort study has been undertaken. The results have been stored in a tidy manner, with the following schema:

column dtype description
n int Number of observations
exposure str Descriptive label indicating category of exposure
disease str Descriptive label indicating category of disease
Note

Note: We use singulars rather plurals for the column titles.

Data was taken from a cohort study analysing the possible association between gestational age and hospitalization for asthma in childhood (Yuan W. et al (2002)).

Recipes

The primary issue here is here is handling multiple contingency tables that model different slices of the data.

Note

Starred recipes require user input.

  1. Load the data*

Prepare the data

StatsModels’s Table2x2 class expects the data in the following shape.

exposure/disease disease (+) no disease (-)
exposed (+) a b
not exposed (-) c d

However, there is no way to guarentee that the data will meet this criteria when we come to slice and reshape it. We solve this by initialising exposures, diseases, two new CategoicalDType variables that will be used to ensure that the data is sorted as expected. We then initialise cat_data, a DataFrame, using data with columns exposure, disease casted to exposures, diseases.

Initialise exposures, diseases as ordered pd.CategoricalDTypes, where:

  • exposures = (exposed…, reference exposure)
  • diseases = (diseased…, not diseased)
Note

The label reference exposure is the nominated neutral exposure to which other exposures will be compared.

exposures = pd.CategoricalDtype(
    categories=['pre-term', 'post-term', 'term'],
    ordered=True
)
diseases = pd.CategoricalDtype(
    categories=['hospitalised', 'not hospitalised'],
    ordered=True
)

Initialise cat_data, a DataFrame, using data with the columns exposure, disease as ordered Categorical variables. Sort cat_data by exposure, disease.

cat_data = (
    pd.DataFrame(
    ).assign(
        n=data['n'].to_numpy(),
        exposure=data['exposure'].astype(exposures),
        disease=data['disease'].astype(diseases)

Initialise the contingency tables

Initialise ctables, a sequence of Table2x2s, where each instance of Table2x2 holds an exposure against the reference exposure.

make_table = lambda df, exposure, ref_exposure: (
    sm.stats.Table2x2(
                df.query('exposure in [@exposure, @ref_exposure]')['n']
                  .to_numpy()
                  .reshape(2, 2)
    )
)
ctables = [make_table(cat_data, exposure, exposures.categories[-1])
           for exposure in exposures.categories[:-1]]

Chi-squared test of no association

Initialise data_arr, an NDArray with shape (r, 2), from column n of of cat_data. Use data_arr to initialise table, a Table.

data_arr = cat_data['n'].to_numpy().reshape(len(exposures.categories), 2)
table = sm.stats.Table(data_arr)

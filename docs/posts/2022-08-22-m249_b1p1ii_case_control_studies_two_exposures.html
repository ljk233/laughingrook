Case-Control Studies With Two Exposures

M249
Statistics
Author

Liam Kelly

Published

August 22, 2022

Summary notes

Aim

Perform an epidemiological analysis on the results of a case-control study with two categories of exposure.

Pre-conditions

A case-control study has been undertaken. The results have been stored in a tidy manner, with the following schema:

column dtype description
n int Number of observations
exposure str Descriptive label indicating category of exposure
casecon str Descriptive label indicating category of case or control
Note

Note: We use singulars rather plurals for the column titles.

The data were taken from a case-control analysing the possible association between political activity and death by homicide was sourced (Mian, A., Mahmood, S.F., et al (2022)).

Notes

We use StatsModelsâ€™s Table2x2 to perform much of the analysis, which expects the data as a sequence with shape (2, 2). This sequence should mirror the following table, where a, b, c, d are integers.

exposure/disease case (+) control (-)
exposed (+) a b
not exposed (-) c d

The primary issue in this collection of recipes is ensuring that the data is ordered as needed to perform the analysis. We cannot guarentee that the data will meet this criteria when we come to reshape it, so we solve this by casting exposure, casecon to ordered pd.Categorical data types.

Prepare the data

Initialise the categories

Initialise exposures, casecons as ordered pd.CategoricalDTypes, where:

  • exposures = (exposed, not exposed)
  • casecons = (case, control)
exposures = pd.CategoricalDtype(
    categories=['attended', 'did not attend'],
    ordered=True
)
casecons = pd.CategoricalDtype(
    categories=['homicide', 'not homicide'],
    ordered=True
)

Initialise a categorical DataFrame

Initialise cat_data, a DataFrame, using data with the columns exposure, casecon as ordered Categorical variables. Sort cat_data by exposure, casecon.

cat_data = (
    pd.DataFrame(
    ).assign(
        n=data['n'].to_numpy(),
        exposure=data['exposure'].astype(exposures),
        casecon=data['casecon'].astype(casecons)
    ).sort_values(
        by=['exposure', 'casecon']
    )
)

Initialise ctable, a Table2x2.

ctable = sm.stats.Table2x2(

Analyse the data

Table with columnar marginal totals

Case-control studies require only the columnar marginal totals, so we drop the row marginal totals.

cat_data.pivot_table(
    values='n',
    index='exposure',
    columns='casecon',
    aggfunc='sum',
    margins=True,
    margins_name='total'
).drop(
    columns='total'
)

Return the results of the chi-squared test.

Note

We pass the argument dtype=object when initialising the Series, so it can handle both float and int data types.

_res = ctable.test_nominal_association()
pd.Series(
    data=[_res.statistic, _res.pvalue, int(_res.df)],
    index=['statistc', 'pvalue', 'df'],
    name='chi-squared test',
    dtype=object
)
